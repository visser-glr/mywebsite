PhpStorm deployment configuration

  • Eerst een ftp user aanmaken op Plesk
  • tools > deployment > configuration...

    • Connection:
    type: ftp
    host: web03.sd-lab.nl Vul de ftp gegevens in die je op plesk heb aangemaakt (user & pw).
    rootpath: "/httpdocs" (map kiezen kan ook) webserver: "https://[studentennummer].stu.sd-lab.nl"

    Mappings:
    localPath: pad naar de map op je computer. (als het goed is ingesteld bij het openen van een project map)
    deploymentPath: Het relatieve pad vanaf het server rootpath (bij connection, dus bv: /mapnaam)
    webPath: Het relatieve pad vanaf de webserver URL (zie connection, dus bv: /mapnaam)

    Het uiteindelijke deploymentPath wordt in dit voorbeeld dus: "/httpdocs/mapnaam"
    Het uiteindelijke webPath wordt in dit voorbeeld dus: "https://[studentennummer].stu.sd-lab.nl/mapnaam"
