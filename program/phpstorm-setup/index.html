phpstorm version 2025.1.1

Opzetten omgeving

  • contoleer waar php is geinstalleerd met $ which php
  • Indien niet aanwezig: php installeren met homebrew
  • php omgeving linken in phpstorm (settings> php > rechter panel)

Variabelen uit andere files zien

  • settings > search: "undefined"
  • click "Undefined variable > options > Search for variables def outside current file"

Automatisch uploaden naar plesk ftp

  • tools > deployment > automatic on save
  • Plesk ftp user aanmaken
  • tools > deployment > config > connection (plesk ftp gegevens invullen> test connection)
  • tools > deployment > config > mappings
