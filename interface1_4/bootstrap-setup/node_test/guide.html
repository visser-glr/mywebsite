Bootstrap met NPM gebruiken

  • Create directory, or with the finder "new folder"
  • cd in directory, cd and drag folder into terminal for location
  • $ cd [dir]
  • Maak package.js met standaard waarden
  • $ npm init -y
  • Installeer bootstrap module@versie
  • $ npm install bootstrap@5.3.6
  • Add js & css links from node_modules folder in index.html (bootstrap & eventueel @popperjs). Zie : load-node.html
  • In case of commit, de node_modules folder is niet nodig te committen. Deze kan opnieuw worden gegenereerd dmv "npm install" in combinatie met package.json.
