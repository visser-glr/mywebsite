Tailwind v4.17 met cli.

* Tailwind scant je classes in de html files en bouwt daar (middels de input.css) zelf de output.css van. Hierdoor gebruik je alleen wat je nodig hebt. De cli zorgt voor de compilatie. Het gebruik van preprocessors (b.v. sass) is daarom eigenlijk overbodig.

Tailwind heeft niet zoals bootstrap een componenten library. Het is meer een verzameling van "utilities" die samen de opmaak verzorgen. Er bestaan wel component libraries voor. (bijvoorbeeld: https://www.material-tailwind.com/)

het cli command voor deze versie:
$ npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch
meer info over cli hier info on preprocessors

* Het tailwind grid. In tegenstelling tot bootstrap krijg je hier wel 4 kolommen als je bijvoorbeeld de klasse "grid-cols-4" gebruikt.
