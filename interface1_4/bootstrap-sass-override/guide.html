Bootstrap met npm

  • Maak een nieuwe folder aan.
  • $ cd [folder name]
  • npm package initialiseren
  • $ npm init -y
  • bootstrap installeren (@specifieke versie)
  • $ npm install bootstrap@5.3.6
  • css en js files linken in index.html

Bootstrap css override

SASS (Syntactic Awesome StyleSheets). of gewoon css?

Sass is een manier om makkelijker css te schrijven omdat je o.a. objecten kunt 'nesten'. makkelijker variabelen kunt gebruiken etc. Het nadeel is dat het gecompileerd moet worden naar css.

Sass installeren

  • vs code install -> live sass compiler
  • vs code: ga naar preferences -> settings en vul verander de settings.formats naar:

        "liveSassCompile.settings.formats": [   

            {
                "format": "compressed",
                "extensionName": ".min.css",
                "savePath": "/css",
            }
        ],

Bootstrap sass override & compileren.

  • Create folder "css" in root. hier wordt straks de gecompileerde minified css gesaved als "main.min.css"
  • Create folder "scss" in de root.
  • Maak een nieuwe file "main.scss" in de scss directory. Hierin importeren we bootstrap.scss (een sass file)
  • In deze file kunnen we ook variabelen overschrijven. (zie main.scss file)
  • selecteer de main.scss file en druk in vs code in de onderste balk op "watch" en save de file.
  • de scss wordt nu gecompileerd naar de css folder.
  • de nieuwe css file (main.min.css) link je nu in de index.html file.
Belangrijk! Open de project folder in vsc zodat de sass compiler weet wat de "root" is. Je kunt in de bootstrap node module eens rondneuzen in de scss folder om de verschillende variabelen te bekijken welke je kunt overriden.

Je kunt uiteraard ook gewoon een css file aanmaken die je later inlaadt dan de bootstrap css file. Hierin kun je dan de gewenste klasse overchrijven zonder sass.
