VsStudioCode-> install extension: live sass compile
Ga naar settings: zoek op "sass" en pas de compile settings aan. Zorg hierbij dat je vanuit een project folder werkt. Anders kan de extensie de root niet bepalen.
Je kunt nu "watch sass" onderaan aanzetten. Bij verandering wordt er nu automatisch gecompileerd.
        {
            "format": "compressed",
            "extensionName": ".min.css",
            "savePath": "/css",
        }
We gaan de primary color waarde overschrijven (zie _variables.scss): 
$primary:       $blue !default;
!default geeft aan: Is er al een waarde opgegeven gebruik die waarde dan voor deze variabele. Zo niet, gebruik dan blauw.
Gelukt!
Je kunt uiteraard ook simpelweg de klasse van de css file overschrijven in een standaard css file zonder gebruik te maken van sass compilatie. Bij grote projecten is het nadeel dat dit minder overzichtelijk is uiteindelijk.
